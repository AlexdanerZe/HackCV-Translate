# HackCV网站文章翻译

HackCV致力于收集、分享最新的机器学习、人工智能和编程技巧相关的内容。建立的初衷是收集真正有料的资讯和博客，整理优秀的代码框架和模型，每天分享自己的学习心得，与大家一起体验这波人工智能对社会生活带来的影响

网站：https://hackcv.com/

翻译已取得站长本人许可，持续翻译ing。。。
